# No More Junk

Premium one-page landing page for NO MORE JUNK.

## Deploy

For GitHub Pages or any static host, upload the files in this repo and set the site entry to `index.html`.

The current `index.html` is self-contained, so it can run without a build step. The React/TypeScript source is included under `src/` for future development.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static server.
